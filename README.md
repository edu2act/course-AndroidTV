# AndroidTV应用开发

>本课程为Android TV应用程序开发课程，主要介绍如何在Android TV环境下开发一款TV应用，以及TV开发的常见问题，通过本课程的学习学生应该能够完成简单TV应用的开发。
本课程是智能设备方向课程体系的高级应用课程。
